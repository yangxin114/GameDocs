🔔 官方更新

3.0.2 版本更新 Build #71841

第13赛季「审判赛季」首个大型修复补丁 · 涵盖资料片内容更新与多项Bug修复

📅 更新日期：2026年5月13日 🌐 适用平台：全平台 📋 Build #71841

🏰 资料片内容更新

📈 护身符更新

  • 套装护身符现在播放独特的掉落音效，以便于辨识。
  • 套装护身符现在使用独特的小地图图标，以提高可见度。
  • 更新了游侠"尼芙尔的锐眼凝眸"护身符3件套奖励的说明，明确指出获得其所有加成的前提是"复仇"状态处于激活中。

🦊 德鲁伊护身符调整

  • "风暴司牧的呼召"5件套奖励现在与2件套的伤害加成叠加，而不是覆盖。
  • "风暴司牧的呼召"2件套奖励现在在增益激活期间提升所有风暴技能伤害，而非仅提升消耗灵力的那个技能。
  • "风暴司牧的呼召"2件套奖励最大加成从75%下调至40%，以补偿其对所有风暴技能的伤害加成。
  • "老狼的印记"现在将其直接伤害加成包含在持续伤害倍率加成中。
  • "老狼的印记"的中毒伤害加成现改为乘法计算，而非加法。

🔄 综合更新

  • 墨菲斯托回响和憎恨回响已加入组队查找器。
  • 新增了各种加载界面提示，提供关于新功能的详细信息。
  • 更新了"多彩调谐棱镜"的说明，明确其可以针对单一抗性。
  • 新增了传送至活跃战争计划的功能。如果没有活跃战争计划，玩家将传送至忒弥斯；否则将传送至当前目标点。

🚧 战役与任务 Bug修复

  • 修复了在"死亡"任务中，完成"对抗墨菲斯托"目标时使用城镇传送门可能导致任务进度卡住的问题。
  • 修复了玩家可能在锁定的或无法进入的战役区域内复活的问题。
  • 修复了重新进入某些区域时，任务NPC对话可能反复触发或重叠的问题。
  • 修复了在"梦之鱼"任务中，石鱼公说话时离开区域可能导致进度卡住的问题。
  • 修复了"谜题之路"任务中错误引用"毁灭"而非"腐化"的问题，该错误可能导致解谜时造成困惑。
  • 修复了"保卫过去"支线任务中，HUD指南针和音频导航辅助功能无法正常工作的问题。
  • 修复了多处隐形墙阻挡任务进度的问题。
  • 修复了"伤口淌血处"任务中玩家死亡后Boss不会重置的问题。
  • 修复了其他多种导致任务进度卡死的问题。
